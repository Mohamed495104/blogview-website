About This Dashboard

This interactive dashboard displays blog posts and user information fetched from the DummyJSON API. You can view posts, filter content, sort data, and interact with various elements to explore the blog ecosystem.

The application demonstrates techniques in DOM manipulation, event handling, data fetching and processing using asynchronous JavaScript, and responsive design principles.

Features:

  • View blog posts with comments
  • Filter and sort content
  • Search functionality
  • User profiles with details
  • Contact form with validation

About DummyJSON API

This project uses DummyJSON.com, a free, open-source API that provides dummy data for testing and prototyping.

The API offers multiple endpoints including posts, users, products, and more with full CRUD functionality.
